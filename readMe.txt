To run frontend:
----------------------------------------------
Prereq: android studio and npm must be installed
----------------------------------------------
1 - go to frontend directory
2 - install npx if not ready installed, run: npm install -g npx
3 - configure a virtual device or connect an android device (follow instructions on android studio). 
if configuring a virtual device, *a device supporting play store must be chosen* . 
3 - run npx react-native start
4 - choose android from the options

To run backend:
------------------------------------------------------------------
Prereq: python, django and a virtual environment must be installed
------------------------------------------------------------------
1. navigate to the same directory as manage.py
2. run python manage.py runserver
